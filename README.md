# BloatBlaster
Automated process to prep computers for customer delivery.

PowerShell script to 

Remove bloatware,
Disable Microsoft startup apps,
Set date/time to Central,
Remove pinned taskbar items,
Install Chrome, Firefox, Adobe Acrobat Reader, and optionally subscription copy of Office 365 Business (Office installation is not working),
Run Disk Cleanup, 
Create Restore Point (named 'Initial Setup'),
Set Power Plan to high performance, disable sleep, disable fast startup, set lid action/power button action (actions not properly setting)

More to come.
